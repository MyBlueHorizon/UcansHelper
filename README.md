### 兼容性警告
为避免出现漏洞，请尽量使用最新版浏览器和 Tampermonkey 使用。
### 当前功能
1. 自动签到  
　　本功能为解决全屏无法显示签到设计，请勿滥用。
2. 查看上课时间
3. 优化课程页面
4. 评论区消息弹幕显示[云课堂播放器改版，暂时弃用]
5. 伪·修改互动标识
　　本功能可以在无麦克风与摄像头情况下将互动标识改为可互动（实际无法正常互动），为解决部分教师忽视学生设备客观条件设计，请勿滥用。
